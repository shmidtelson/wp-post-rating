# Wordpress post stars rating plugin
## Description ##
WP-POST-RATING is powerful rating plugin with ajax security requests.
Features:
* Very faster
* Seo-friendly
* without jQuery (Native js)

## Requirements ##
* Wordpress >= 4.9.8
* PHP >= 7.2
